Use Twitter API to gather 'Likes' and 'Retweet' data for each tweet¶

To gather this data I used the Python library Tweepy to access the Twitter API.

The Twitter API required authorisation before it could be used. As such, I had to create a Twitter account and apply for a developer account. Once my application for this was approved, I was able to generate the Consumer API keys, the Access Token and the Access Token Secret that were required for authorisation.

Please note that I have removed these tokens before submitting this report.

api = tweepy.API(auth, wait_on_rate_limit=True)
